# Remove-ScriptSession 
 
Removes a persistent Script Session from memory. 
 
## Syntax 
 
Remove-ScriptSession -Id &lt;String&gt; 
 
Remove-ScriptSession -Session &lt;ScriptSession&gt; 
 
 
## Detailed Description 
 
Removes a persistent Script Session from memory.
